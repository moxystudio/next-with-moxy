ESLint & Stylelint

The boilerplate includes our own linting presets for linting both Javascript and CSS files.

For linting Javascript files, we use eslint-config-moxy, and for linting CSS files we use stylelint-config-moxy.

ℹ️ Make sure that your editor supports ESLint and Stylelint for the best experience.
