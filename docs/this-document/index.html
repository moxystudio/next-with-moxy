This document

This document is the default documentation for next-with-moxy projects.

Keep in mind, if you have received a project developed with next-with-moxy, you have received a version of this documentation tailored to your project. Though much of the information in this document is shared between projects, we recommend you always check the documentation sent with your project for more accurate and tailored instructions.

The first time you do this, you must install the dependencies of the documentation. From the folder of the project, run:

npm install --prefix docusaurus

Afterwards, and every time you need to access the documentation again, you can run:

npm run docs

This script will automatically open your documentation in your default browser.
